# PPT模板目录

## 使用说明

将您的高质量PPT模板文件（.pptx格式）放置在 `user_templates/` 目录下。

## 模板要求

1. **格式**: 必须是 `.pptx` 格式（PowerPoint 2007及以上版本）
2. **版式**: 模板应包含多种常用版式（标题页、内容页、对比页、数据页等）
3. **母版**: 建议使用PowerPoint的母版功能定义统一的配色和字体
4. **占位符**: 版式中应包含清晰的占位符（标题、正文、图片等）

## 推荐模板来源

- [优品PPT](https://www.ypppt.com) - 高质量商务模板
- 自己设计的企业模板
- 第三方专业设计师模板

## 示例

```
templates/
└── user_templates/
    ├── business_pro.pptx        # 商务专业风格
    ├── tech_modern.pptx         # 科技现代风格
    └── minimal_elegant.pptx     # 简约优雅风格
```

## 注意事项

- 模板文件名建议使用英文，避免中文和特殊字符
- 确保模板文件完整且未损坏
- 建议模板文件大小不超过50MB
